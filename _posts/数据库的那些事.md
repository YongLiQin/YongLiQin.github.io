###　管理数据库

### 添加用户

#### MySql

```shell
create user `user`@`host` identified by `password`;
```

#### MongoDB

```shell
db.createUser({user: "", pwd: "", roles: ["readWrite"]})
```

### 更改密码

#### mysql

```shell
mysqladmin -u root -p password
```

#### mongodb

```shell
db.update(<username>, {
		pwd: <password>,
		roles: [{role: <>, db: <dbname>}]
	});
```
